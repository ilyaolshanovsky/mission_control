# Дашборд «Школа 21» + ассистент Ольга

Интерактивный дашборд сети кампусов с чат-ботом **Ольга** в правом нижнем углу.

Ольга отвечает по данным дашборда (KPI, кампусы, зоны, график стартов) и умеет формировать сводные отчёты в Excel.

## Запуск

```bash
cd shkola21-dashboard
cp .env.example .env   # укажите GIGACHAT_CREDENTIALS или OPENAI_API_KEY
chmod +x run.sh
./run.sh
```

Откройте http://127.0.0.1:8010/

## Как это устроено

- `dashboard.html` — лендинг/дашборд + виджет чата
- `data/campus_data.json` — данные кампусов для контекста ассистента
- `server/app.py` — FastAPI: отдаёт дашборд и `/api/chat`
- `server/context.py` — собирает релевантный контекст из JSON
- `server/prompts.py` — системный промпт Ольги

## Примеры запросов

- «Сколько кампусов в красной зоне?»
- «Сравни Москву и Казань по MAU»
- «Сформируй отчёт по франшизам»
- «Когда старт бассейна в Липецке?»

При отчёте появится кнопка **Скачать отчёт (XLSX)** — файл с листами: Сводка, Топ MAU, KPI по кампусам, Выводы.

## Автообновление из Google Таблицы

Данные можно подтягивать из Google Sheets без ручной правки JSON.

### 1. Подготовьте таблицу

Сделайте лист с теми же колонками, что в выгрузке дашборда (**Выгрузить все кампусы XLSX**), минимум:

`Кампус`, `Вид кампуса`, `Адрес`, `Рабочих мест`, KPI-поля (`MAU`, `CSI рег. заказчика` и т.д.).

График стартов и история периодов пока берутся из локального `data/campus_data.json` и **дополняются** при синхронизации по имени кампуса.

### 2. Опубликуйте CSV-ссылку

В Google Таблице: **Файл → Поделиться → доступ по ссылке** (читатель).

Ссылка на экспорт CSV:

```text
https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/export?format=csv&gid=SHEET_GID
```

`gid` — id вкладки (в URL таблицы после `#gid=`).

### 3. Настройте `.env`

```env
GOOGLE_SHEET_CSV_URL=https://docs.google.com/spreadsheets/d/.../export?format=csv&gid=...
SHEET_SYNC_INTERVAL_SEC=300
```

### 4. Перезапустите сервер

```bash
./run.sh
```

Сервер:
- при старте скачивает таблицу;
- обновляет каждые `SHEET_SYNC_INTERVAL_SEC` секунд;
- отдаёт свежие данные в `/api/data` (дашборд и Ольга).

Ручное обновление: `POST /api/sync`  
Статус: `GET /api/sync/status`

### Альтернатива для закрытых таблиц

Если таблица не публичная — подключите **Google Sheets API** + service account (отдельная интеграция). Для прототипа проще CSV-экспорт по ссылке.

## Открыть только HTML без сервера

Можно открыть `dashboard.html` напрямую в браузере — дашборд работает, но чат потребует запущенный backend на порту 8010.
